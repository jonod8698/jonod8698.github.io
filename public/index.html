Wpad Security

JD  published on 2022-03-20 included in  category Security
What is WPAD?WPAD stands for Windows Proxy Auto-Discovery. It is a protocol to obtain the URL for a PAC file. WPAD uses the following methods in order to discover the PAC file URL: DHCP DNS WINS LLMNR NetBIOS Hosts Lmhosts The PAC file tells the host where to direct network traffic. This is usually used to force traffic through a web proxy. Why is it insecure?When DHCP or DNS is misconfigured or not configured, an attacker can provide the client with a PAC file that directs traffic to a compromised server.

Remote Credential Guard

JD  published on 2022-03-19 included in  category Security
SummaryRCG protects admin credentials by ensuring they are not passed over the network to the target device. DetailsClients with RCG enabled do not send primary credentials to the target machine. This mitigates the risk of stolen credentials if the target is compromised. In RCG, the client LSA acts as a reverse proxy for ticket requests from the target. Hence, the target never needs to store primary credentials. RCG will not allow NTLM fallback.
